## Шаг 1: Получи бесплатный API ключ AssemblyAI

1. Зайди на https://www.assemblyai.com/
2. Нажми **Sign Up** (бесплатно)
3. Подтверди email
4. В Dashboard скопируй **API Key** (выглядит как: `abc123def456...`)
5. Бесплатно даётся **$50 кредитов** — это ~333 часа аудио

---

## Шаг 2: Задеплой на Netlify (5 минут)

### Вариант A: Drag & Drop (самый простой)

1. Зайди на https://netlify.com → **Sign up** (бесплатно)
2. После входа нажми **"Add new site" → "Deploy manually"**
3. Перетащи **всю папку `voicescript`** прямо в браузер
4. Сайт уже работает! Получишь ссылку вида `random-name.netlify.app`

### Вариант B: Через GitHub (лучше для обновлений)

1. Создай аккаунт на github.com
2. Создай новый репозиторий и загрузи все файлы
3. На Netlify: **"Import from Git"** → подключи репозиторий
4. Build settings оставь пустыми (статический сайт)
5. Deploy!

---

## Шаг 3: Добавь API ключ на Netlify (ОБЯЗАТЕЛЬНО)

> Это делается ОДИН РАЗ. Ключ хранится на сервере — пользователи его не видят.

1. В Netlify Dashboard зайди в свой сайт
2. **Site settings → Environment variables → Add variable**
3. Заполни:
   - **Key:** `ASSEMBLYAI_KEY`
   - **Value:** твой ключ от AssemblyAI (например `abc123def456`)
4. Нажми **Save**
5. Зайди в **Deploys → Trigger deploy → Deploy site** (нужно для применения переменной)

**Готово!** Теперь все пользователи могут транскрибировать бесплатно.

## Как это работает (архитектура)

```
Пользователь загружает файл
        ↓
index.html (браузер)
        ↓ fetch('/.netlify/functions/transcribe')
netlify/functions/transcribe.js  ← здесь хранится ASSEMBLYAI_KEY
        ↓
AssemblyAI API (реальная транскрипция)
        ↓
Результат → пользователю
```

Ключ никогда не покидает сервер Netlify — пользователи его не видят!
